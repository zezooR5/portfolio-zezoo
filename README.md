## How to Use

1. Open `index.html` in your web browser
2. Navigate through sections using the sidebar
3. Customize content in `index.html` to match your information
4. Replace `images/profile.jpg` with your own photo
5. Add tech stack icons to `images/icons/` folder

## Image Requirements

For best results with the image styling:
- **Profile Image**: 800x800px or larger, square aspect ratio
- **Tech Icons**: 256x256px, PNG format with transparency
- **Project Images**: 800x400px, landscape orientation
